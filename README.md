# node-shopping-cart
## Introduction
完整購物網站雛形
包含:
- 登入/登出
- 購物車
- 帳單
- 簡易信用卡金流
## Use Tool
- nodejs
- express
- bootstrap
- handlebarsjs templet 
- mongodb
- passport
## Build Setup
使用前需有mongodb，並執行
``seed``中的``product-seeder.js``匯入產品 即可
```
# install dependencies
npm install

# serve start at localhost:3000
npm start
```
